# 2026-06-18

## opener

Thursday. ReconVLA is the paper to read today: it empirically documents that current VLAs allocate visual attention uniformly (dispersed), not to task-relevant regions, and proposes reconstruction of the "gaze region" as a training signal to correct it. That is the empirical motivation surf has been assembling from first principles — now it has a peer-reviewed anchor. The spec should lead with ReconVLA's finding alongside SEE-SAL.

Separately, When Does Language Matter? is a new precision result: language sensitivity in VLA manipulation is step-specific, with 30-50% performance drop under non-English and sharp variation across manipulation steps. If language influence is step-dependent, surf's contrastive scorer may need to be step-adaptive — a design constraint not yet in the spec.

## papers

### digest (auto)

**core**
- [ReconVLA: Reconstructive Vision-Language-Action Model as Effective Robot Perceiver](https://www.semanticscholar.org/paper/c87ea2afcd9212e5128e01daf17c733d285ac345) — Song, Zhou, Zhao et al. · arXiv
  Empirically confirms that VLA visual attention is dispersed — not grounded on task-relevant targets. Proposes gaze-region reconstruction as an implicit grounding signal: a diffusion transformer reconstructs the task-relevant image region conditioned on the VLA's visual outputs. Directly validates surf's motivating problem and operationalizes [[eval/reconstruction]] as a training objective rather than just an evaluation proxy. Differentiation question: ReconVLA uses reconstruction without geometric priors or contrastive language scoring — what does surf's structural prior add over reconstruction-based implicit grounding?

- [When Does Language Matter? Multilingual Instructions Reveal Step-wise Language Sensitivity in VLAs](https://www.semanticscholar.org/paper/094964f7e202bcf536d92558d0e490ed231c8c34) — Dong, Han, Niu et al. · arXiv
  Language sensitivity in VLA manipulation is highly non-uniform across steps: certain steps are strongly language-dependent (language-sensitive steps), others are language-agnostic (motor primitive steps), with 30-50% success rate drop under non-English instructions. Critical for [[claims/language helps]]: if language influence is step-specific, surf's instruction-conditioned visual selection should be step-adaptive, not fixed at instruction time. This is a direct design constraint for the contrastive scorer that the current spec has not addressed.

- [Native Active Perception as Reasoning for Omni-Modal Understanding](https://arxiv.org/abs/2606.19341v1) — Xing, Xu, Wang et al. · arXiv
  OmniAgent: formulates video understanding as a POMDP-based iterative Observation-Thought-Action cycle, where the agent selectively requests audio-visual observations based on query difficulty rather than scanning all frames. This is surf's selective-perception thesis at video-frame granularity with an explicit POMDP formulation. Relevant to [[questions]] (active perception): how does surf's within-image patch selection compose with OmniAgent's between-frame selection? If the POMDP framing applies across granularities, it provides a principled unification for the Tianqin active-perception connection.

**adjacent**
- [Feat2Go: Visual Feature-Grounded Value Estimation for Embodied Reinforcement Learning](https://www.semanticscholar.org/paper/d0ff6f94082e1e125459f7a240c4eb536a18b428) — Shu, Lin, Wei et al. · arXiv
  Derives a continuous manipulation progress signal from patch-level visual world-model similarity, addressing sparse reward in VLA RL. If the progress signal is grounded in specific visual patches, then selective visual attention is a prerequisite for reliable value estimation — surf's patch selection problem is not independent of the RL training objective but coupled to it. Relevant to surf's efficiency/RL module and to [[claims/iterative selection]] as a credit assignment angle.

- [Vision Transformers Need Better Token Interaction](https://www.semanticscholar.org/paper/ce02cd3ecf85e6f7872055908017b00391534a0d) — Su et al. · Semantic Scholar
  Identifies "semantic diffusion" in ViTs: global semantic information spreads through patch tokens beyond what is locally justified during prolonged training, degrading patch-level spatial specificity for dense prediction. If surf's encoder undergoes semantic diffusion, patch representations lose the local spatial discriminability that the contrastive scorer and line-guided selection assume. May motivate using early-layer or lightly fine-tuned ViT features for the selection signal rather than final-layer representations.

- [Colosseum V2: Benchmarking Generalization for Vision Language Action Models](https://www.semanticscholar.org/paper/ff60e668c280b09db7875904a8f2e768261f899c) — Morgan, Vijay, Oh et al. · arXiv
  Large-scale VLA generalization benchmark (60 tasks × 20 distractor conditions) decomposing performance across visual, spatial, and language distribution shifts. For surf's evaluation: if token pruning degrades task success, Colosseum V2's axis decomposition would identify which failure mode dominates — visual-shift sensitivity, spatial-shift sensitivity, or language-shift sensitivity — giving more diagnostic information than a single success-rate scalar.

**horizon**
- [STARE: Surprisal-Guided Token-Level Advantage Reweighting for Policy Entropy Stability](https://arxiv.org/abs/2606.19236v1) — Luo, Sun, Wu et al. · arXiv
  In LLM RL post-training (GRPO), decomposes per-token entropy into trajectory-level advantage × token-level surprisal, using surprisal as a credit-assignment weight. The surprisal signal — how unexpected a token is relative to the current policy — is an information-theoretic token importance measure. Conceptually analogous to surf's visual-token scoring: informative patches are surprising relative to the structural prior, which could operationalize the selection criterion information-theoretically. Horizon: LLM domain, but the surprisal-as-importance framing may transfer.

- [Does VLA Even Know the Basics? Measuring Commonsense and World Knowledge Retention in VLAs](https://arxiv.org/abs/2606.19297v1) — Kachaev, Moskalenko, Skripkin et al. · arXiv
  Introduces Act2Answer: VLA knowledge benchmarks where the model must answer through object-placement actions. Finds significant commonsense and world knowledge loss after manipulation fine-tuning. Horizon concern for surf: if VLA fine-tuning degrades the pretrained language backbone, surf's language-conditioned visual selection is grounded in impoverished representations — a compounding risk on top of [[claims/language helps]] brittleness noted in When Does Language Matter?

## questions

- ReconVLA gaze region: how is the "gaze region" defined in ReconVLA — eye-tracking labels, saliency proxy, or derived from the model's own attention map? If gaze region comes from human eye-tracking data, the pipeline overlaps with [[eval/gaze]] and surf's gaze supervision plan. If attention-map derived, the circular-dependency concern applies: the model defines what to reconstruct.
- Language sensitivity step structure (When Does Language Matter?): does the step-dependency pattern correlate with manipulation phases (approach, grasp, release) or with specific token content in the instruction? If phase-correlated, surf's contrastive scorer should be conditioned on the phase or timestep signal, not just the language instruction.
- OmniAgent POMDP granularity: OmniAgent selects frames in video; surf selects patches within a frame. Does the POMDP formulation compose across granularities (frame-level then patch-level within a frame), or does patch-granularity require a different observation-action space definition?
- Feat2Go patch similarity: does the world-model progress signal use all patches or only task-relevant ones? If all patches contribute equally to the similarity score, visual-token pruning would directly distort the value estimate — surf's selection is a confound for the reward signal, not just an efficiency gain.
- Semantic diffusion layer: at which depth does semantic diffusion dominate in standard ViTs? If it emerges late in the network, using intermediate-layer features for surf's contrastive scorer would preserve spatial discriminability without requiring a separate encoder.
