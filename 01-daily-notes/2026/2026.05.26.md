## papers

### digest (auto)

**core**
- [AVA-VLA: Improving Vision-Language-Action models with Active Visual Attention](https://www.semanticscholar.org/paper/bf94018d5eff3065895cb856eef16181bbe4f4e9) — Xiao et al. · arXiv
  History-aware attention: conditions visual token selection on prior-timestep observations rather than treating each step as independent. Addresses the non-Markovian perception gap that [[questions]] flags under "architecture" — how past observations should inform current patch selection. A baseline that [[claims/structure helps]] must be compared against once extended to action tasks.

- [Gaze on the Prize: Shaping Visual Attention with Return-Guided Contrastive Learning](https://www.semanticscholar.org/paper/ac43acda376c1d2a4c79f236046064b9bb062df8) — Lee, Chuang, Gao et al. · arXiv
  Return-guided contrastive learning identifies which image regions are task-relevant in visual RL — the agent learns where to attend by contrasting high- and low-return trajectories. The most concrete template for the contrastive-alignment experiment (action item from 2026-05-24 meeting); also directly tests whether task-relevant regions can be recovered without an explicit structural prior. If return signal is sufficient, [[claims/structure helps]] needs to argue why a structural shortcut is preferable to learning from returns.

- [ResPrune: Text-Conditioned Subspace Reconstruction for Visual Token Pruning in LVLMs](https://www.semanticscholar.org/paper/d8d6855727d59b8f55efe9cade6191b073037d6a) — Li, Zheng, Liang et al. · arXiv
  Training-free token pruning: tokens whose features cannot be reconstructed from a text-conditioned subspace are pruned. Bridges [[claims/language helps]] (language conditioning identifies useful tokens) and [[eval/reconstruction]] (reconstruction residual as relevance signal); the subspace formulation is a cleaner alternative to the MAE-residual metric that avoids the reconstruction-artifact confound. Read alongside ReconVLA.

**adjacent**
- [Option Discovery Using LLM-guided Semantic Hierarchical Reinforcement Learning](https://www.semanticscholar.org/paper/b635dd85b8edb9bf2daaf7dc62840c78c1fe4c94) — Shek, Tokekar et al. · arXiv
  LLMs decompose tasks into subgoals that define options in the hierarchical RL framework (LDSC). Directly addresses Dr. Lee's suggestion to read hierarchical control/options (2026-05-24 meeting). Concrete template for how task decomposition could gate which features matter at each sub-goal — the framing surf needs to connect its efficiency/RL module to the options literature.

- [CORE: Compact Object-centric Representations as a New Paradigm for Token Merging in LVLMs](https://www.semanticscholar.org/paper/adddedced7c22beac87851012a82985a909b3b52) — Lei, Wang, Lee et al. · arXiv
  Object-centric grouping before merging: tokens belonging to the same semantic object are merged rather than arbitrary attention-similar pairs. Challenges [[claims/structure helps]] at the granularity level — structure (object boundaries) may matter more as a grouping criterion than as a patch-selection criterion; if object-level is the right unit, patch-level line-guided selection is solving the wrong sub-problem.

- [Non-Markovian Long-Horizon Robot Manipulation via Keyframe Chaining](https://www.semanticscholar.org/paper/e24f15946a4a18c63b3c7215dbc5b0c38bbe07ee) — Chen, Tan, Zhu et al. · arXiv
  Selects keyframes from prior observations and chains them to handle non-Markovian VLA tasks. Orthogonal axis to surf's within-step spatial selection: temporal selection of which past frames to attend to vs. spatial selection within a frame. Useful for scoping the efficiency/RL module — does it need to handle temporal selection, spatial selection, or both?

**horizon**
- [Learning in Low-Dimensional Subspaces: Orthogonal Bottlenecks for Reinforcement Learning](https://arxiv.org/abs/2605.26012v1) — Todorov, Sabatelli et al. · arXiv 2026
  Fixed-rank orthogonal bottleneck in RL networks forces policy and value representations into low-dimensional subspaces matching the task's intrinsic dimension. Task-relevant visual structure may be intrinsically low-dimensional — this is a representation-level prior parallel to surf's structural priors. Speculative bridge: line/shape structure might work because it forces token selection into the low-dimensional subspace where the task actually lives.

- [Seeing but Not Believing: Probing the Disconnect Between Visual Attention and Answer Correctness in VLMs](https://www.semanticscholar.org/paper/25a72b7f78aac9b015ce0e29e215f2bc495678ae) — Liu, Chen, Liu et al. · arXiv
  VLMs can attend to the correct visual evidence and still answer incorrectly — perception and reasoning decouple. Empirical warrant for the [[eval/gaze]] falsifier ("gaze alignment improves while task performance does not"): attention metrics may grade perception, not downstream task competence. Before trusting any gaze-alignment metric as a proxy for action relevance, this decoupling needs to be addressed.

## questions
- AVA-VLA conditions on observation history: is the history aggregation done in the token domain (each past patch contributes separately) or at a compressed representation? If token-level, this compounds the compute problem rather than reducing it — the budget framing changes.
- Gaze on the Prize: contrastive return signal identifies task-relevant regions without a structural prior. Does the contrastive signal converge before the RL signal (return) is noisy early in training? If not, the prior module may still be needed as a warm-start.
- ResPrune uses text-conditioned subspace reconstruction training-free: which text representation? CLIP-style frozen embeddings or instruction-conditioned? The language conditioning in [[claims/language helps]] requires instruction-specificity, not just category-level text.
- CORE merges by object group rather than patch similarity — but "object group" requires an object detector or segmentation step. What is the compute cost of that step relative to the savings from merging? Is this cheaper than running the full dense ViT?
