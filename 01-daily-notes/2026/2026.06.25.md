## opener

Thursday. GSPA unread — day 12. Contrastive alignment spec unwritten — 7+ weeks.

Three papers today land directly on the spec's differentiation table. DepthCache uses depth gradients as a geometric signal to guide training-free VLA token merging — this is the strongest single-feature structural-prior baseline yet, and the spec must answer what line-drawing structure adds over depth. CogVLA operationalizes language conditioning at the computation-routing level rather than the patch-selection level: instruction drives which modality features are processed, not which spatial tokens survive. The distinction matters — if routing-level language conditioning subsumes patch-level conditioning, surf's contrastive scorer needs a sharper justification. TTF-VLA (AAAI) solves the temporal coherence axis: tokens stable across consecutive frames are fused rather than re-processed; tokens that change are retained. This is an operational "when to not re-select" criterion that addresses [[claims/iterative selection]] without language conditioning — and raises the question of whether surf's scorer adds signal beyond temporal coherence alone.

In the adjacent tier, When Attention Collapses provides the clearest external validation of surf's two-stage argument yet: early attention is structure-biased, late attention is semantic-biased, and single-metric pruning collapses in one dimension. The paper's fix — structural selection first, semantic refinement after — is surf's prior + contrastive scorer in token-pruning form. The challenge it raises: if stage-aware attention pruning already captures both stages implicitly, what does surf's explicit geometric prior add?

## papers

### digest (auto)

**core**
- [DepthCache: Depth-Guided Training-Free Visual Token Merging for VLA Inference](https://www.semanticscholar.org/paper/349a9a94f5d47624a65a84e9125fd430f18f6fea) — Li, Ma, Ding, Zhu · arXiv
  Uses monocular depth maps to identify spatially complex regions (high depth gradient) and retain tokens there; merges tokens in low-complexity depth-uniform areas. Training-free. This is the most direct single-feature baseline for surf's structural prior in VLA: depth-guided merging addresses the same "which tokens to keep" question with depth rather than line-drawing structure. The spec must state explicitly what 2D line structure adds over depth — depth captures surface discontinuities, lines capture within-surface contours and shape edges; these are distinct and the empirical overlap on manipulation scenes is unknown.

- [CogVLA: Cognition-Aligned VLA via Instruction-Driven Routing & Sparsification](https://www.semanticscholar.org/paper/e30e7b2e70556078190f63ac50f613a2f1efd6b3) — W. Li, Zhang, Shao, He, Nie · arXiv
  Language instruction drives computation routing through the VLA backbone, sparsifying which modality features are active at each step. Operationalizes [[claims/language helps]] at a coarser granularity than surf: CogVLA conditions which computation paths run; surf conditions which spatial patches enter those paths. Differentiation question: are routing-level and patch-level language conditioning compositional (CogVLA decides the computation subgraph; surf decides which visual tokens reach it) or competitive (both reduce VLA compute along the same bottleneck)?

- [TTF-VLA: Temporal Token Fusion via Pixel-Attention Integration for VLA](https://www.semanticscholar.org/paper/02462f0f53d1b48867e35eae8901d75996e5a662) — Liu, Zhang, Li, Zhou, Wu · AAAI
  Identifies tokens whose pixel-level attention is stable across consecutive manipulation frames and fuses rather than re-processes them; retains high-change tokens. Accepted at AAAI. The "when to not re-select" criterion from [[claims/iterative selection]] — without language conditioning. Key question: do the frames where TTF-VLA's temporal coherence breaks correspond to the language-sensitive steps identified by Dong et al. (6/23 digest)? If yes, coherence is a cheap proxy for language-sensitivity. If no, language adds non-redundant signal.

**adjacent**
- [When Attention Collapses: Stage-Aware Visual Token Pruning from Structure to Semantics](https://www.semanticscholar.org/paper/63620ae17342d270cc0e01328de8f15726f38637) — Wang, Zhang, Han, Zhang · Semantic Scholar
  Diagnoses "attention collapse" in single-metric token pruning: early attention captures structure (edges, contours), late attention captures semantics; methods that use only one collapse in the other. Fix: stage-aware policy — structural pruning first, semantic refinement after. Surf's prior + contrastive scorer is this two-stage policy made explicit: the line prior is the structural stage, the contrastive scorer is the semantic stage. External validation of surf's architectural argument. Counter-challenge: if implicit stage-aware attention achieves both without an explicit geometric prior, does surf's prior add anything beyond forcing the structural stage to be geometrically specific rather than attention-generic?

- [ReconVLA: Reconstructive VLA as Effective Robot Perceiver](https://www.semanticscholar.org/paper/c87ea2afcd9212e5128e01daf17c733d285ac345) — Song, Zhou, Zhao, Chen, Ding · AAAI
  Current VLAs let visual attention disperse rather than focus on task-relevant regions; adds reconstruction auxiliary objective to force grounded attention during training. Accepted at AAAI. Addresses the same dispersed-attention failure that surf's structural prior targets, but through training-side supervision rather than inference-side selection. Baseline framing: if reconstruction auxiliary objective alone recovers grounded visual attention, does surf's explicit prior add anything at test time? The answer likely depends on whether reconstruction grounds attention on target objects generically or on the geometrically relevant features specifically.

- [Spatial Forcing: Implicit Spatial Representation Alignment for VLA](https://www.semanticscholar.org/paper/263896525561a4c4299dcf1bf80815088d00e509) — F. Li, Song, Zhao, Wang · arXiv
  VLAs built on 2D-pretrained VLMs lack 3D spatial awareness; spatial forcing loss aligns internal representations to spatial structure without 3D supervision data. Training-time spatial grounding vs. surf's inference-time spatial selection — different entry points for the same spatial-awareness gap. These are composable: spatial forcing improves the representation quality that surf's selector operates on.

**horizon**
- [Geometry Matters: 3D Foundation Priors for Semantic Correspondence](https://www.semanticscholar.org/paper/7e8ad4aaee7c2ed3942348872daed6ae4e66099a) — Jesslen, Dunkel, Kortylewski · Semantic Scholar
  3D geometric priors from foundation models (DINOv2 features, monocular depth) improve 2D semantic correspondence by anchoring visual matching in 3D structure. Horizon framing: surf's line-drawing structural prior is a lightweight 2D instantiation of the same "geometry as reference frame for visual matching" hypothesis. If 3D geometric priors improve correspondence across images, 2D structural priors should improve within-image patch selection for the same mechanistic reason — structural features are the stable reference frame that semantic signals attach to.

- [How and What to Imagine? Visual Thinking for Cross-View Spatial Reasoning](https://www.semanticscholar.org/paper/0693a362cd67937bfd6a1b1cd836d5a1c74703d4) — Q. Yang, Sikarwar, Le, Zhang, Shi · Semantic Scholar
  VLMs fail at cross-view spatial reasoning with language-only CoT; interleaving imagined visual states (visual thinking) into the reasoning chain recovers performance. Horizon connection: if spatial reasoning requires explicit visual intermediate representations, then surf's patch selection is the prerequisite for efficient visual thinking — you want the model to reason about geometrically relevant patches, not full-resolution noise. Positions selective visual computation as upstream of visual CoT rather than a standalone efficiency trick.

## questions

- DepthCache vs. line prior: Do depth-gradient-dense patches correspond to line-drawing-dense patches on manipulation scenes? If they largely overlap, depth is a cheaper geometric prior and surf must argue the incremental value of 2D line structure. If they diverge (depth = surface discontinuities, lines = within-surface contour/shape), they are orthogonal priors and surf addresses structure that depth misses.
- CogVLA routing vs. surf's selection: Are instruction-driven computation routing (CogVLA) and language-conditioned patch selection (surf) compositional or competitive? If compositional, surf's scorer could run on top of CogVLA's routing — the active computation subgraph receives the task-relevant token subset.
- TTF-VLA coherence vs. language-sensitive steps: Do the frames where TTF-VLA's temporal coherence breaks align with the language-sensitive steps from Dong et al.? If yes, coherence is a cheap proxy for language-sensitivity and surf's scorer is redundant at stable frames. If no, language adds signal not captured by temporal coherence alone.
- When Attention Collapses structural stage: Does the paper's structural stage (edge/contour-based pruning) select the same patches as surf's line-drawing prior on manipulation scenes? If yes, surf's novelty concentrates entirely in the semantic (language-conditioning) stage and the VLA integration.
- ReconVLA: Does the reconstruction auxiliary objective improve grounded attention generically (all informative regions) or geometrically specifically (the structurally dense regions surf's prior targets)? This determines whether ReconVLA is a complete alternative or a complementary approach.
