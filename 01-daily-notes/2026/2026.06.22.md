## papers

### digest (auto)

**core**
- [Seeing but Not Believing: Probing the Disconnect Between Visual Attention and Answer Correctness in VLMs](https://www.semanticscholar.org/paper/25a72b7f78aac9b015ce0e29e215f2bc495678ae) — Liu et al. · Semantic Scholar
  Fifth data point on where surf's contrastive scorer should operate: shallow layers focus on text, deep layers sparsely attend to localized evidence — but attention and answer correctness decouple even when the right tokens are attended. Surf implication: patch selection is necessary but not sufficient; the scorer must be applied at the layer where attended visual features are actually decoded into answer-relevant representations. Joins One Layer's Trash (6/19), Finetuning Fewer Layers (6/20), Visual Instruction Tuning Aligns (6/20), Hidden Evolution (6/21).

- [Differentiable Efficient Operator Search](https://www.semanticscholar.org/paper/034c48d33dc3bf4c29cc50608c219b749c862d6f) — Pei et al. · Semantic Scholar
  Pruning, merging, pooling, and adaptive reweighting are recast as regimes of a shared differentiable operator space; searches jointly over where/how-much/how to reduce tokens. For the [[experiments/contrastive alignment]] spec: surf's structural-prior + contrastive-scorer is a task-conditioned geometric regime not covered by existing operators in this space — which sharpens the novelty argument. The joint-search training framework also subsumes the LightVLA and Beyond Surrogate Gradients training-objective arguments into a single design decision.

**adjacent**
- [Bring My Cup! Personalizing Vision-Language-Action Models with Visual Attentive Prompting](https://www.semanticscholar.org/paper/c8702d791db3820d55cec32531fd2a043f24a762) — Lee et al. · Semantic Scholar
  Instance-level language-conditioned VLA visual attention via few-shot reference images, no contrastive training. Nearest non-contrastive baseline for [[claims/language helps]]: VAP achieves instruction-conditioned visual grounding without end-to-end geometric priors or contrastive training. The spec must explain what surf's structural prior and contrastive training add over few-shot visual prompting — the answer is likely: generalization to novel objects/instructions without reference images.

- [MemoryWAM: Efficient World Action Modeling with Persistent Memory](https://arxiv.org/abs/2606.20562v1) — Yang et al. · arXiv (Jun 2026)
  Efficient world action model that resolves the bounded-window/efficiency tradeoff in WAMs via persistent memory across manipulation steps. Adjacent to [[claims/iterative selection]]: persistent memory addresses the same multi-step visual evidence problem as iterative selection but at world-model granularity (stored visual states) rather than patch granularity (on-demand refinement). If MemoryWAM's retained tokens cluster at structural features across steps, that is independent evidence for [[claims/structure helps]].

- [STARRY: Spatial-Temporal Action-Centric World Modeling for Robotic Manipulation](https://www.semanticscholar.org/paper/f134eaaeb2cf6d4ac3526993199033b7f858a35b) — Tian et al. · Semantic Scholar
  World-model-enhanced VLA that jointly denoises future spatial-temporal latents and actions, creating an action-centric spatial representation. Alternative answer to surf's core question about what makes a visual patch task-relevant: STARRY says patches that predict future action-relevant spatial structure. If action-centric prediction converges on the same patches as surf's line prior on manipulation scenes, the prior may be a cheap proxy for STARRY-style future prediction.

**horizon**
- [Exploratory Experience Shapes the Geometry of Predictive Representations](https://www.semanticscholar.org/paper/32fbfdc2a3af37f998167a40607656c9af6be023) — Shilova et al. · Semantic Scholar
  Active sensing as an action-perception loop where exploration shapes the geometry of internal predictive representations via predictive coding. Theoretical grounding for surf's open question on active perception: patches selected iteratively should maximize reduction in uncertainty about action-relevant state — which is a predictive-coding criterion, not just a reconstruction criterion. Connects surf's iterative selection to the active perception framing in Tianqin's project at a formal level.

Horizon note for surf's evaluation planning: Colosseum V2 is becoming a standard generalization evaluation for VLA claims; surf's structural prior should be tested against it to demonstrate that efficiency gains do not come at the cost of generalization.

## questions

- Seeing but Not Believing layer gap: at what LLM layer depth does attention-to-correctness coupling emerge in Liu et al.'s analysis? Does the gap close at intermediate layers (matching Visual Instruction Tuning Aligns) or only final layers? The answer constrains which layer surf's contrastive scorer should tap — if the gap closes at intermediate layers, early extraction is both sufficient and preferred.
- Differentiable Efficient Operator Search: does the unified operator space include task-conditioned geometric inductive biases, or only operators defined over visual similarity and attention scores? If geometric operators are excluded, surf's prior is a new operator regime not yet covered — strengthening the spec's novelty argument.
- Bring My Cup! / VAP mechanism: does VAP apply few-shot reference images by direct feature matching (retrieve-and-highlight) or by modifying the VLA's attention weights? If attention-weight adaptation, the mechanism is closer to surf's contrastive scorer than it appears — and the delta surf adds (geometric prior + end-to-end training) becomes the core differentiator.
- MemoryWAM token statistics: which tokens are retained in MemoryWAM's persistent memory across manipulation steps? If retained tokens cluster at object boundaries and geometric features, this validates [[claims/structure helps]] independently of surf's explicit prior.
