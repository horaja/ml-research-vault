Wednesday, July 22. Contrastive alignment spec unwritten — 16 weeks. GSPA unread — day 39. "When Attention Collapses" (STS) unread — day 28. Visual Access Boundaries unread — day 8.

**ThinkProprio** (Wang et al., today's digest) is the most direct design-space competitor yet: it routes visual token selection via *discretized proprioceptive state* rather than language instruction or structural prior. The diagnostic before writing the spec's method section: does robot state alone co-locate with contact-geometry patches on LIBERO? If yes, surf's design needs explicit ablation of each component (state vs. state + structure vs. state + structure + language). If state and structure select overlapping regions, the structural prior may be approximating a signal already recoverable from proprioception — reframing its contribution as improving selection quality under noisy state, not injecting new information.

**Beyond Attention or Similarity** (Zhang et al.) introduces a diversity criterion for token pruning. If diversity-selected tokens outperform attention- and similarity-selected tokens, surf's contrastive structural scorer — which concentrates selections on high-edge-density regions — may be choosing a correlated cluster rather than a maximally informative set. Relevant ablation: diversity oracle vs. structural prior on the same visual budget.

Write the spec.

## questions

- ThinkProprio × surf design: ThinkProprio discretizes robot proprioceptive state into a visual grounding signal without language or structural prior. On LIBERO contact tasks, does state routing co-locate with surf's line-density heatmap? Key temporal asymmetry: structural prior selects contact-geometry patches *before* contact (from visual structure alone); proprioceptive state only reveals contact *after* it. If structural and state selections diverge pre-contact, the prior provides information the state signal cannot.
- Diversity criterion × surf scorer: If surf's structural scorer selects edge-dense patches, those patches may cluster in feature space (contact-geometry features correlate). Does diversity maximization over the full visual token set produce qualitatively different selections from structural scoring on LIBERO scenes? If yes, the diversity criterion is a separate competing baseline. If no, structural scoring implicitly maximizes diversity in the relevant feature directions.
- CLIP texture bias × surf encoder: Hernández-Cámara et al. show CLIP's texture bias deepens over training. If surf's contrastive scorer uses a CLIP vision encoder, does line-density prior injection work *against* the encoder's texture bias (selecting edge patches the encoder ranks low) or *with* it (edge patches happen to also be textured)? This determines whether the prior is complementary to or fighting the encoder.
- Masked visual actions × world model pretraining: If Fei-Fei/Wu world model pretraining from video learns contact-region structure without explicit supervision, surf's structural prior at inference may add limited value in a well-pretrained system. Empirical test: does surf's prior improve over a world-model-initialized VLA baseline, or does the improvement collapse when the backbone is better pretrained?
- V4 evidence convergence × [[claims/v4 relevance]]: Willeke et al. (DNN-driven cell characterization), @jiangComputationalConstraintsUnderlying (SOM model), and @wangLargescaleCalciumImaging2024 (calcium imaging) all support V4 topographic organization. Does three-way convergence mean the V4 dispersity measure is robust enough to operationalize as a per-patch prior, or do the three methods measure different aspects of topography that don't all support the same selection criterion?
- Teaching VLAs × spec motivation: Yang and Chen find that VLA training on semantic VQA/CoT omits fine-grained spatial supervision in driving. Is the same gap present in manipulation training data (LIBERO, Open-X)? If manipulation demonstrations include explicit contact-keyframe labels, the training-time spatial gap is smaller and inference-time structural priors have less work to do.
