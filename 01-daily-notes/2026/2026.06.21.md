## papers

### digest (auto)

**core**
- [SpecPrune-VLA: Accelerating Vision-Language-Action Models via Action-Aware Self-Speculative Pruning](https://www.semanticscholar.org/paper/e5e4c5f50fef44a34d1235659e2cdb6cf3db5a6b) — Wang, Xu et al. · arXiv (2026)
  Existing per-step VLA pruning loses >20% success rate by ignoring global context; SpecPrune uses action-aware context across steps to guide token removal. Sixth direct efficiency prior work for the spec's differentiation table (CogVLA, LightVLA, ElegantVLA, AC²-VLA, MoLe-VLA, SpecPrune-VLA). Key question: does SpecPrune's "action context" overlap with surf's step-adaptive language conditioning — or are action history and language instruction orthogonal signals?

- [MoLe-VLA: Dynamic Layer-skipping Vision Language Action Model via Mixture-of-Layers for Efficient Robot Manipulation](https://www.semanticscholar.org/paper/5b2142f267139a69eb1d9c598ae3291fead7e4e7) — Zhang, Dong et al. · AAAI 2026
  Mixture-of-Layers router decides which LLM layers to skip per step in a VLA, avoiding the final-layer semantic loss that kills early-exit approaches. Adaptive compute at layer granularity rather than token granularity — complementary to surf's spatial token selection. Together they define a two-axis efficiency space (layer × token) that should appear explicitly in the spec's design-space section. Not previously in digest.

- [TTF-VLA: Temporal Token Fusion via Pixel-Attention Integration for Vision-Language-Action Models](https://www.semanticscholar.org/paper/02462f0f53d1b48867e35eae8901d75996e5a662) — Liu, Zhang et al. · AAAI 2026
  Training-free temporal merging of redundant visual tokens across consecutive VLA frames via pixel-attention integration, preserving spatial structure while reducing per-step overhead. Third axis for the efficiency space alongside surf's spatial selection and MoLe's layer skipping. Composability question: does TTF's temporal merging operate before or after surf's spatial selection, and do they compete for the same score signal?

**adjacent**
- [SPOT-E: Test-Time Entropy Shaping with Visual Spotlights for Frozen VLMs](https://arxiv.org/abs/2606.20244v1) — Yin, Hu et al. · arXiv (Jun 2026)
  Test-time visual intervention: spotlights localized decisive evidence regions and verifies whether highlighting actually reduces decoder entropy, rather than open-loop highlighting. No retraining. Interesting contrast to surf's training-time contrastive scorer: if entropy reduction identifies task-relevant patches at inference without training, it is a zero-cost baseline for surf's selection signal. The entropy-reduction criterion is also a candidate unsupervised training signal for [[claims/efficiency without collapse]].

- [The Hidden Evolution of Disguised Visual Context inside the VLM](https://arxiv.org/abs/2606.20077v1) — Suharitdamrong, Alex, Awais et al. · arXiv (Jun 2026)
  Controlled study of in-context vs. layer-injection visual token integration in VLMs: the two architectures produce qualitatively different visual representation dynamics across LLM layers. Directly relevant to the layer-depth question — which architecture the surf backbone uses determines where the contrastive scorer should extract features. Fourth convergent data point on this question alongside One Layer's Trash (6/19), Finetuning Fewer Layers (6/20), and Visual Instruction Tuning Aligns through Abstraction (6/20).

- [Don't Blind Your VLA: Aligning Visual Representations for OOD Generalization](https://www.semanticscholar.org/paper/5444dd08c9b57e0c4dc9d26464af5d2908aa92b6) — Kachaev, Kolosov et al. · AAMAS 2026
  Systematic study showing that VLA fine-tuning degrades the visual representations inherited from VLM pretraining — naive action fine-tuning causes representation collapse, harming OOD generalization. Directly motivates surf's structural prior from an alignment argument: if VLA fine-tuning compresses structurally informative pretraining features, an explicit geometric prior re-surfaces them. Pairs with the superposition hypothesis from Sparsity, Superposition, and Forgetting (6/20 durable). Same first author as Does VLA Even Know the Basics? (6/18) — building a coherent picture of what fine-tuning destroys.

**horizon**
- [π0.5: a Vision-Language-Action Model with Open-World Generalization](https://www.semanticscholar.org/paper/1e2a82ef5e0325a26ca0344c4f8c256c52fae7ec) — Physical Intelligence (Black, Finn, Levine et al.) · arXiv (2026)
  Frontier generalist VLA from Physical Intelligence: heterogeneous co-training across robot morphologies and task types for open-world generalization. Horizon anchor — sets the current VLA frontier that surf's efficiency claims must eventually be positioned against. If scale and co-training diversity already produce task-relevant visual attention implicitly, surf must show what explicit structural priors add beyond scale.

- [Beyond Surrogate Gradients: Fully Differentiable Token Pruning for Vision-Language Models](https://www.semanticscholar.org/paper/5865585d29f0c9c83b717729ec3f82edc705bed4) — He, Yao et al. · arXiv (2026)
  Gumbel-Softmax surrogate gradients for discrete token selection are unreliable because optimization follows surrogates rather than the true selection process; proposes true differentiable selection. Relevant to surf's training objective: if the contrastive scorer drives discrete patch selection, the gradient estimator choice matters. LightVLA (6/19) trains selection end-to-end — this paper provides the training-level argument for why differentiable > surrogate for surf's implementation.

## questions

- SpecPrune action context: what specifically is the "global context" SpecPrune uses across steps — compressed action history, cumulative attention map, or instruction embedding? If action-step history, does it handle instruction-sensitive steps differently from motor-primitive steps? If instruction embedding, it directly overlaps with surf's step-adaptive contrastive scorer as motivated by When Does Language Matter? (6/18).
- MoLe router mechanism: does MoLe's Mixture-of-Layers gate use a confidence signal from early layers (early-exit style) or a learned input-conditional router? If learned router, what features condition it — action step, visual content, or instruction? A router conditioned on both instruction and visual content would be a layer-level analogue of surf's contrastive scorer.
- TTF temporal fusion: TTF is training-free — does it use pixel similarity or attention similarity to identify temporally redundant tokens across frames? If attention-based, temporal fusion and spatial selection may compete for the same score signal — composability depends on whether they operate in the same representation space.
- SPOT-E entropy criterion: can SPOT-E's entropy-reduction criterion (spotlight patches → decoder uncertainty decreases) be adapted as an unsupervised training signal for surf's contrastive scorer? If decisive-evidence patches are precisely those that reduce decoder entropy, this gives an information-theoretic grounding for patch selection without requiring gaze supervision.
- Don't Blind architecture: does Kachaev et al.'s alignment method preferentially preserve structurally informative features (edges, geometry) or all VL representations equally? If geometric features are disproportionately degraded by action fine-tuning, the argument for surf's structural prior strengthens considerably.
- Carried from 6/20: STS repulsion mechanism (feature-space vs. spatial grid); CogVLA instruction routing mechanism; PosPrune bias correction magnitude; EventVLA event trigger type; GazeLNN training-data specificity; VLA fine-tuning layer-criticality for continuous-control vs. contact-rich tasks.
