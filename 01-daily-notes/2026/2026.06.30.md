Zhang & Bisk coin "instruction blindness" for the failure mode where limited-data VLA finetuning causes policies to ignore language instructions in favor of visual shortcuts. This is surf's language conditioning claim stated as a failure mode — if standard finetuning reliably induces instruction blindness, surf's contrastive scorer is the targeted intervention for the cases where geometric structure disambiguates which patches carry the task-relevant language signal. ## questions

- Instruction blindness × geometric prior: does the visual shortcut that causes instruction blindness concentrate specifically on object-category features (texture, color) rather than spatial-structural ones (contact zones, shape)? If yes, surf's line-drawing prior breaks the shortcut by suppressing texture/color while preserving the spatial-structural signal the instruction requires.
- Visual attention sink × line prior: do the sink-accumulating tokens (low-attention-reward) in VLMs co-locate with line-sparse regions (flat backgrounds, featureless surfaces)? If sink location and structural sparsity co-locate, the geometric prior prevents the sink as a direct byproduct of structural selection — a tighter mechanism claim than "structural patches matter."
- Willeke et al. V4 functional map vs. Jiang et al. SOM model: does the DNN-characterization V4 map show the same shape/texture dispersity column structure as the computational model? If yes, dispersity scores can be derived from a DNN V4 model directly, bypassing the digital-twin pipeline.
- Declarative/procedural decoupling × LA4VLA: if declarative skill transfer is handled by language-action pretraining without vision, does surf's structural prior add value only on the procedural side (spatial perception of contact zones, tool engagement)? If yes, this is a cleaner framing of surf's contribution than "improves VLA broadly."
