## papers

### digest (auto)

**core**
- [GSPA: Geo-Semantic Prior-Guided Attention for Long-Horizon Robot Manipulation](https://www.semanticscholar.org/paper/593fe3f7bbdcbd6169f9903fcab28e8e65316c4e) — Che, Wang, Wen et al. · arXiv
  Uses geometric and semantic priors jointly to guide VLA attention through long-horizon manipulation rollouts, where standard VLA attention drifts across stage transitions. This is the most directly overlapping published work with surf's core framing to date: both geometric structure and language-semantic conditioning applied to VLA attention, in a manipulation context. The spec's related-work section needs to characterize what GSPA does that surf does not (or vice versa) before claiming novelty for the combined prior.

- [FlashVLM: Text-Guided Visual Token Selection for Large Multimodal Models](https://www.semanticscholar.org/paper/4fddca70a71254c39d619c4f53727b00b4fea9fe) — Cai, Zhang, Yang et al. · arXiv
  Proposes a "shallow prefill" phase that computes stable text-to-image relevance scores from early VLM layers rather than deep attention maps, then selects visual tokens accordingly. The instability argument against deep-attention-based selection is important for [[experiments/contrastive alignment]]: surf's contrastive scorer also avoids relying on the final LLM attention, but for different reasons (we want geometric structure, FlashVLM avoids instability). FlashVLM's shallow-prefill design is a concrete language-conditioned selection baseline that doesn't require explicit contrastive training — a stronger ablation than "language replaced by zeros."

- [TTF-VLA: Temporal Token Fusion via Pixel-Attention Integration for Vision-Language-Action Models](https://www.semanticscholar.org/paper/02462f0f53d1b48867e35eae8901d75996e5a662) — Liu, Zhang, Li · AAAI
  Training-free temporal structural prior: fuses historical frame tokens into current frame tokens via pixel-attention weighting, exploiting frame-to-frame coherence in manipulation tasks to reduce effective token count at inference without retraining. An orthogonal structural prior to surf's spatial/geometric approach. If TTF-VLA's temporal prior achieves comparable token compression to geometric priors at no training cost, [[claims/structure helps]] needs a controlled comparison: temporal vs. geometric vs. combined.

**adjacent**
- [Beyond Sight: Finetuning Generalist Robot Policies with Heterogeneous Sensors via Language Grounding](https://www.semanticscholar.org/paper/3267c7120cb67167d0fd3d65faac0ac6e5a24abd) — Jones, Mees, Sferrazza, Abbeel et al. · arXiv
  Language-grounded finetuning of robot policies on heterogeneous sensor modalities (vision + touch + audio), with language as the shared grounding signal that bridges modality gaps. Adjacent to surf's language-conditioning design: the paper demonstrates that language is a viable bridge between structurally different input modalities — the same mechanism surf uses to bridge structural priors with task-relevant patch scoring. From Oier Mees (tracked author — should have arrived via author feed).

- [MoLe-VLA: Dynamic Layer-skipping Vision Language Action Model via Mixture-of-Layers for Efficient Robot Control](https://www.semanticscholar.org/paper/5b2142f267139a69eb1d9c598ae3291fead7e4e7) — Zhang, Dong, Zhang et al. · arXiv
  Treats each LLM layer as an "expert" and learns which layers to activate per input, orthogonal to token pruning. Efficiency axis: skip layers rather than tokens. If layer-skipping achieves similar speedups to token pruning for VLA deployment, the spec's motivation for token-level spatial selection (as opposed to layer-level skipping) needs to address the tradeoff: spatial selectivity (surf/LightVLA/CogVLA) vs. depth selectivity (MoLe-VLA). The two are composable but their interaction is unexplored.

- [STARRY: Spatial-Temporal Action-Centric World Modeling for Robotic Manipulation](https://www.semanticscholar.org/paper/f134eaaeb2cf6d4ac3526993199033b7f858a35b) — Tian, Jin, Yu et al. · arXiv
  Jointly denoises future spatial-temporal latents and action sequences, tightly coupling spatial-temporal prediction with action generation. Adjacent to surf's open question of whether world-model predictions could guide spatial token selection: STARRY's joint denoising implies that the spatial prediction and the action prediction share a latent, which is a different path to task-relevant spatial attention than surf's contrastive scoring approach.

**horizon**
- [MiniMax Sparse Attention](https://arxiv.org/abs/2606.13392v1) — Lai, Xu, Yang et al. · arXiv Jun 2026
  Blockwise sparse attention for 1M+ context: a lightweight Index Branch scores KV blocks, selects only high-scoring blocks for full attention, discards the rest. The Index Branch → sparse selection → full compute pipeline is architecturally analogous to surf's patch scorer → token selection → VLA backbone pipeline. If the Index Branch design (a small scorer that estimates which blocks matter without running full attention) generalizes to spatial patch scoring in VLA contexts, it provides a training-free architecture that surf's contrastive scorer should benchmark against.

- [HYDRA-X: Native Unified Multimodal Models with Holistic Visual Tokenizers](https://arxiv.org/abs/2606.13289v1) — Zhang, Qiu, Cui et al. · arXiv Jun 2026
  Unifies image and video tokenization within a single ViT, addressing temporal coherence and semantic awareness simultaneously. Horizon: surf's patch selection operates on ViT patch tokens; if the tokenization layer changes (e.g., unified spatiotemporal tokens rather than spatial-only patches), the patch-level structural prior needs to be re-specified. HYDRA-X's design is an early signal of where visual tokenization is heading — worth monitoring to avoid building on a tokenization assumption that the field is moving away from.

## questions

- GSPA geometric prior: which geometric features does GSPA use — depth, surface normals, line drawings, or spatial coordinates? If depth or line drawings, the overlap with [[claims/structure helps]] is direct and must be addressed as the nearest related work in the spec, not a generic "prior methods" reference.
- FlashVLM shallow prefill: does the "shallow prefill" use the VLM's own early layers, or a separate lightweight model? If it uses early VLM layers, FlashVLM's mechanism is structurally similar to surf's contrastive scorer but with text-image similarity (CLIP-style) rather than contrastive training. The ablation question: does contrastive fine-tuning outperform zero-shot shallow-layer text-image similarity for token selection?
- TTF-VLA temporal granularity: at what level does TTF-VLA operate — spatial patch tokens, or higher-level visual embeddings? If spatial patch tokens, the temporal prior and surf's spatial geometric prior operate on the same representation, and combining them is a natural extension.
- MoLe-VLA interaction with token pruning: does dynamic layer-skipping change which tokens are important at remaining layers? If skipped layers would have reranked tokens (as Reroute suggests), then MoLe-VLA may exacerbate surf's hard-pruning fragility by skipping exactly the layers where token importance reversals occur.
