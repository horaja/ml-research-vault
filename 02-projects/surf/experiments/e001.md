# e001

## hypothesis
Line-guided patch selection can reduce visual compute without reducing ImageNet-10 classification accuracy.

## setup
- Source: [[sources/line-biased-vision-encoder]]
- Repo commit inspected: `0543ccf`
- Model: `SelectiveMagnoViT`
- Task: ImageNet-10 validation classification
- Prior/selector: line drawing density with spatial weighting
- Baseline: full patch budget
- Metric: Top-1, Top-5, GFLOPs

## result
Repo README/docs report:
- full patch budget: Top-1 `0.734`, Top-5 `0.964`, `1.408` GFLOPs
- line-guided selective low compute: Top-1 `0.758`, Top-5 `0.968`, `0.448` GFLOPs

## interpretation
This result motivates the prior module and the broader structural-prior claim.

## failure analysis
- Missing repeated seeds and uncertainty intervals.
- Missing center-bias and Canny edge-density comparisons in the durable note evidence.
- The result could be dataset/model-specific.
